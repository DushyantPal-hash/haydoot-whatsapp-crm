# HeyDoot WhatsApp CRM

An AI-powered WhatsApp CRM platform designed to help businesses manage customer conversations, automate workflows, improve team collaboration, and boost sales productivity — all from a single dashboard.

---

## 🚀 Features

### 💬 Shared Team Inbox
- Manage multiple WhatsApp numbers
- Multi-agent support
- Internal team collaboration
- Assign conversations to agents
- Real-time messaging

### 🤖 AI Automation
- Auto-replies
- AI-generated responses
- Smart ticket creation
- Workflow automation
- Rule-based triggers

### 📊 Analytics & Reports
- Agent performance tracking
- Response time analytics
- Chat activity insights
- Customer engagement reports
- Export reports

### 🎫 Ticketing System
- Auto-ticket generation
- SLA tracking
- Priority management
- Status updates
- Customer query management

### 📢 Broadcast Campaigns
- Bulk messaging
- Scheduled campaigns
- Audience segmentation
- Template messaging

### 🔒 Security & Access
- Role-based permissions
- Secure authentication
- Session management
- Contact masking

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/DushyantPal-hash/haydoot-whatsapp-crm.git
```

## 2️⃣ Move Into Project

```bash
cd haydoot-whatsapp-crm
```

## 3️⃣ Setup Database

- Create a MySQL database
- Import the SQL file

## 4️⃣ Configure Environment

Update your database credentials inside:

```bash
/config/config.php
```

Example:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'haydoot_crm');
define('DB_USER', 'root');
define('DB_PASS', '');
```

## 5️⃣ Run Project

Move project into your server directory:

### XAMPP

```bash
htdocs/
```

### WAMP

```bash
www/
```

Then open:

```bash
http://localhost/haydoot-whatsapp-crm
```
